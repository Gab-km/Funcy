# Funcy

Funcy は C#/VB 用の関数型プログラミングライブラリです。

Funcy は [NuGet](https://www.nuget.org/packages/Funcy/) からインストールできます:

```ps1
PM> Install-Package Funcy
```

## ドキュメント

* [クイックスタート](QuickStart.md) から始めましょう。
